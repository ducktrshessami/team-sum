# team-sum

Not to be confused with dim sum.

A CLI program to quickly create a team employee summary page based on user input.

Powered by Node.js

# Installation

After installing [Node.js](https://nodejs.org/) and cloning this repository, run the following command in the package directory:

```
npm install
```

# Usage

Just run the following command in the package directory:

```
node .
```

# Testing

Run the following command in the package directory:

```
npm run test
```

Testing is powered by [Jest](https://www.npmjs.com/package/jest)
